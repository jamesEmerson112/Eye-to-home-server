# Eye-to-home-server

## Backend Setup (Flask)

### 1. Create and activate the virtual environment
```bash
python3 -m venv venv
source venv/bin/activate
```

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

### 3. Run the Flask app
```bash
python app.py
```

The app will be available at [http://127.0.0.1:5000](http://127.0.0.1:5000)
